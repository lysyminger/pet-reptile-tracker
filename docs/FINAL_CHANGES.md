## 📊 统计逻辑

### 1. 宠物数量
```javascript
// 查询当前用户的宠物数量
db.collection('pet_info')
  .where({ user_openid: openid })
  .count()
```

### 2. 喂食次数
```javascript
// 查询所有宠物的喂食记录总数
const petIds = pets.map(p => p._id);
db.collection('feed_logs')
  .where({ pet_id: db.command.in(petIds) })
  .count()
```

### 3. 目前体重
```javascript
// 查询所有宠物中最新的体重记录
db.collection('weight_logs')
  .where({ pet_id: db.command.in(petIds) })
  .orderBy('record_date', 'desc')
  .limit(1)
  .get()
// 返回 weight 字段
```

### 4. 喂养天数
```javascript
// 从第一只宠物创建到今天的天数
const startDate = new Date(最早宠物的 created_at);
const now = new Date();
const daysCount = Math.ceil((now - startDate) / (1000 * 60 * 60 * 24));
```
